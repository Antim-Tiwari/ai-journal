# AI Journal System

AI Journal System is a full-stack web application that allows users to write journal entries, analyze emotions using AI, and view insights based on their writing patterns.

The application combines modern frontend technologies with backend APIs and AI-based text analysis.

---

## Features

- Create journal entries
- Select ambience (Forest, Ocean, Mountain)
- AI-based emotion analysis
- Extract keywords from journal entries
- View insights based on past entries
- Clean responsive UI built with Tailwind CSS

---
